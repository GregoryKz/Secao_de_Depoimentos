Seção de Depoimentos em HTML e CSS

Este projeto consiste em uma seção de depoimentos responsiva, criada em HTML e CSS, para exibir feedbacks de clientes ou usuários de maneira organizada e visualmente atraente. A seção valoriza cada depoimento com fotos, nomes, cargos e citações, garantindo credibilidade e destaque.

Funcionalidades

Layout em grade responsiva para diferentes tamanhos de tela.

Cartões de depoimento estilizados com borda superior e sombra suave.

Fotos circulares dos autores com borda discreta.

Título da seção centralizado e chamativo.

Efeito hover nos cartões, proporcionando interação visual.

Tipografia e cores modernas e legíveis.

Tecnologias Utilizadas

HTML5 para a estrutura da seção.

CSS3 para estilização, efeitos e responsividade.

Estrutura do Projeto
/assets
  └─ css/
       └─ style.css       # Estilos da seção de depoimentos
  └─ img/
       └─ image3.jpeg     # Fotos dos clientes/depoimentos
index.html                 # Página com a seção de depoimentos
README.md                  # Documentação do projeto

Uso

Abra o arquivo index.html em qualquer navegador moderno.

Personalize os depoimentos, fotos, nomes e cargos conforme necessário.

Ajuste cores, fontes ou bordas no CSS para se adequar ao estilo do seu site.
